# Module 4 – Analytics Dashboard

## Overview
This module provides analytics and visualization features for the Smart Home Energy Management System (SHEMS).

It converts raw energy consumption data into meaningful insights for both users and administrators through dashboards and reports.

---

## Purpose of the Module
Energy data alone is not useful without interpretation.  
This module analyzes the data generated by the Energy Tracking module and presents it in a clear and actionable format.

---

## Core Responsibilities

### User Analytics
- Weekly energy consumption trends
- Monthly usage trends
- Device-wise energy comparison
- Peak usage day detection
- Energy saving recommendations

### Admin Analytics
- Total system energy consumption
- Top energy consuming devices
- Identification of high usage users
- System-wide energy trends
- Smart system recommendations

## Main Components

### AnalyticsController
Provides analytics dashboard for users:
- Weekly energy trend
- Monthly trend
- Device comparison
- Energy tips

### AdminAnalyticsController
Provides system-wide analytics for administrators:
- Total energy usage
- Top devices
- High usage users
- System recommendations

### AnalyticsService
Handles user-level analytics calculations.

### AdminAnalyticsService
Handles system-wide analytics calculations.

---

## Integration with SHEMS
This module consumes data from:
- Device Management Module
- Energy Tracking Module

The generated insights are used by:
- Users for monitoring and optimization
- Admin for system-wide decision making
