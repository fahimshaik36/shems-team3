# Smart Home Energy Management System (SHEMS)

## Module 1: Authentication and Role-Based Access Control (RBAC)

This module implements secure authentication and role-based authorization using Spring Boot and Spring Security.

👉 [View Module 1 Authentication & RBAC PDF](./docs/Module1_Authentication_RBAC.pdf)

## Overview
- User registration and login
- Encrypted password storage (BCrypt)
- Role-Based Access Control (RBAC)
- Role-based redirection (USER / ADMIN)

## Tech Stack
- Java, Spring Boot, Spring Security
- Thymeleaf, Hibernate/JPA
- MySQL

## How to Run
1. Create database `role_auth`
2. Configure `application.properties`
3. Run the application
